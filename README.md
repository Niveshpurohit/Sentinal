# Sentinel: Python-Based Network Packet Sniffer

A lightweight network analysis tool built with Python and Scapy to monitor real-time traffic and identify potential security threats.

## 🚀 Features
- **Live Capture:** Monitors incoming and outgoing IP traffic.
- **Protocol Identification:** Decodes TCP, UDP, and ICMP protocols.
- **Security Alerts:** Basic detection for ICMP (Ping) sweeps and SYN patterns.
- **Minimal Footprint:** Optimized for low memory usage during long-term sniffing.

## 🛠️ Tech Stack
- **Language:** Python 3.10+
- **Library:** Scapy
- **Environment:** macOS / Linux (Requires Root Privileges)
